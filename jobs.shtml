Jobs

Indianapolis is currently experiencing a Java developer shortage.

If you are looking for java work, it should not be too hard to find it.

If you are looking for java workers, that may be more of a challenge.

There are other Java User Groups that are fairly close to Indianapolis that serve their communities as we do.
If you can't find a job here in Indy or you can't find workers, the other groups do have mailing lists as well that can be leveraged.

It is common to have to sign up to a list before you can post to it.

Some of these groups have multiple mailing lists, so please visit their sites if you are interested in more than just job postings.

If you want to search a little wider than that, you can consult java.net.
In addition, there are other online communities such as java.meetup.com that may yield interesting things.
