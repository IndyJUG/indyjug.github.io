New to Java?

Thinking in Java is a good place to start, but if you have little programming experience, even TIJ can seem a bit daunting. While not for everyone, some people will really enjoy Sams Teach Yourself Java 2 in 24 Hours which has been made available for free by the publisher.

Another good resource for learning Java is Thinking in C: Foundations for Java & C++ (local). Knowing something about C will make you a better Java programmer. This program is designed to teach you C with the thought that you will turn around and tackle Java (or C++).
